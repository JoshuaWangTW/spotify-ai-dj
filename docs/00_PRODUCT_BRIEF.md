# 00 Product Brief — Spotify AI DJ

## One-liner
一個支援手機與 PC 的 PWA 音樂助理：使用者用自己的 Spotify Premium 播放音樂，AI 像 DJ 一樣推薦、銜接、介紹古典與爵士樂。

## Target user
- 已有 Spotify Premium。
- 想聽古典或爵士，但希望有人導聆。
- 想在工作、烘豆、開車、店內營業、晚上放鬆時快速取得合適歌單。

## Core value
Spotify 已經能播放音樂，但不一定會「說明為什麼這樣排」或「教你怎麼聽」。此產品的價值是：
- 情境理解。
- 曲目銜接邏輯。
- 古典 / 爵士導聆。
- 個人偏好記憶。

## Product positioning
不是 Spotify clone。是 AI Music Companion / AI DJ Layer for Spotify。

## MVP success criteria
使用者輸入一句話，例如：
> 我想聽爵士，想學一點，不要太硬。

系統能：
1. 產生 5 首候選曲或搜尋 query。
2. 解釋推薦邏輯。
3. 播放或加入 queue。
4. 每首提供短導聆。
5. 記住使用者回饋。
